This zip file contains the materials for the extension command

STATS FIND FILE

The command defines an IBM SPSS Statistics file handle for the location
where the file is found.

This material requires at least 
SPSS Statistics 17 and 
the Python plug-in.

It includes a dialog box definition, a help file, the syntax definition, 
and the implementation file.

To install this procedure, 
1. Unzip all of the files into the extensions subdirectory of your 
SPSS Statistics installation.

2. In SPSS Statistics, use
Utilties>Install Custom Dialog
to add this item to the menus.

In version 18, the menu path is 
Utilities>Custom Dialogs>Install Custom Dialog

The dialog box will appear on the File menu.

Executing
STATS FIND FILE/HELP.
will display the complete syntax help.

This command was previously known as IBMSPSS FIND FILE.

Questions and comments should be directed to the
